**Reseptisovellus**

Kuvaus:
Reseptisovellus on full-stack sovellus, jossa on mahdollista selata, lisätä ja poistaa reseptejä. 

Rekisteröitynyt käyttäjä pystyy kirjautumaan sisään ja ulos, lisätä ja poistaa omia reseptejä sekä tykätä haluamistaan 
resepteistään. 

Respetisivulla käyttäjä pystyy järjestämään reseptejä kategorioittain sekä hakemaan reseptejä hakutoiminnolla. Lisäksi käyttäjä
pystyy tarkastelemaan yksittäisten reseptien tietoja. Profiilisivulla näkyy rekisteröityneen käyttäjän tiedot sekä lista hänen 
lisäämistään ja tykkäämistään resepteistä. Rekisteröitynyt käyttäjä pystyy halutessaan poistamaan itse lisäämiään reseptejä ja oman tilinsä.

Lisäksi löytyy kontaktisivu, jonka kautta voi ottaa yhteyttä sovelluksen ylläpitäjään. Tätä toimintoa ei kuitenkaan ole
vielä toteutettu valmiiksi.

Teknologia:
- kieli: JavaScript
- frontend: React, React Router, Tailwind CSS, Bootstrap, Cloudinary
- backend: Node.js, Express, MongoDB Atlas (Mongoose)
- autentikaatio: JSON Web Token
- RESTful API Requests: Axios
